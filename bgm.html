Custom BGMs

Several BGMs are available to be played on actual matches, but you can use your own in some modes.

How to:
Inside the installation directory open the "Sound" folder.
You find an additional "MyBGM" folder inside that folder.
Put your musics here. Only OGG Vorbis format is supported.

Also it is possible to put png pictures here with the same name to be displayed during BGM selection.

Example:
 windom.ogg --- music file
 windom.png --- image file
