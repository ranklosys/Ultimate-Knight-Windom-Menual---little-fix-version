Get started

You can launch the game by double-clicking on the WindomXP.exe located in the "Game" folder.



Adjust the game's settings to match your PC's performance on the start-up screen.
You can use the Simple settings. It will automatically configure some options.

※Recommended to set "Medium quality" setting at first.
※See the system requirements. If your PC reach the "Recommended" performance, you may use the Highest quality.
　If your PC's performance only meets with the "Minimum", set the settings below.

Resolution Set the size of the screen.
Texture quality 0 is the highest possible. The quality will be decreased
with the larger number. The game's speed will be improved.
Vision quality Specifies the view distance. The higher number increase
the distance. The lower distance improve the game's speed.
Max frame skip rate This option will prevent the game become slow. It will
attempt to keep the game's speed by reducing the displayed frame count.
It Specifies the maximum amount of the skipped frames. Usually use 5-10.
Do not use shader The shader function will improve the 3D objects' quality.
Disabling it will improve the game's speed.
※The shader functions requires a Pixel Shader 2.0 compatible video card.
HDR Make the 3D objects' to reflect the dinamic lights
To use this option, the shader must be enabled.
Shadow Determines the shadows of the units' projected to the background.
The lower shadow quality will improve the game's speed.
To use this option, the shader must be enabled.
Sound setting Set the volume of each types of the sounds.
It has no effects on the game's speed.
Key config window Set the keys and its functions in the opened window.
You can use keyboard or other types of controllers.
With the use of the JavaScript keycodes in the Config.ini, you can edit the movement keys.
