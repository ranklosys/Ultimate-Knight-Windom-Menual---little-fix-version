Custom pilots

Ultimate Knight Windom XP comes with three pilots, but you can edit and
use your own pilots.

The custom pilots edited by yourself also can be used in online matches.
You don't need to provide the custom data to the other players.
■How to create
1. Use pilot editor
Ultimate Knight Windom XP PowerUp Kit comes with a PilotEditor software.
With the use of this software, the pilot editing is relatively easy.
You can create two different pilot versions.

2. Pilot Version 1
This is the simple version of the pilot, easy to create.
Use Ace Nuko and Lieutenant Nuko from the base pilots as example.
There are minimum four step to create a new pilot.

Procedure
　1: Open the "Pilot" folder inside the game's folder.
　2: Create a new folder here, and name it (feel free to use any name).
　3: Create a pilot.txt file, and write the pilot's name inside it.
　　　※Don't use line breaks.
　4: Provide a picture for the pilot, name it to face.png.
　　　※The picture must to be in PNG format.
　　　※It must be 200 x 150 width and height.
　　　※If its size larger than 100 Kb, the picture will not be visible during online matches.

Displayable instant messages
　5: Create 0-9.txt inside your pilot's folder, and write the messages in those.
　　　※For example 0.txt, 1.txt, 2.txt., and so on.
　　　※The messages will be displayed corresponding to the 0-9 keys.
　　　※It is not require to create all (0-9). Create only those which you need.

Adding sounds to those messages
　6: Provide the sound files you want to play, name it to 0-9.ogg.
　　　※For example 0.ogg, 1.ogg, 2.ogg., and so on.
　　　※This will be played alongside with the display of the message when you press the 0-9 key.
　　　※Only OGG Vorbis sound files can be played.
　　　※If its size is larger than 50 Kb, the sound will not be audible during online matches.

Adding voice effects to be played during the match
　7: Prepare the sound files and name like below. 　
	atk1-5.ogg : Playing while you attack
	s_atk1-5.ogg : Playing while you use stronger attacks
	dmg1-5.ogg : Playing while you receive damages
	destroy1-5.ogg : Playing while you destroys an opponent
	die1-5.ogg : Playing while your unit has been destroyed
	rebirth1-5.ogg : Playing while you respawn
	wake1-5.ogg : Playing when you must to get up
　
　　　※The 1-5 sounds will be played randomly in the definited situations.
　　　※If its size is larger than 50 Kb, the sound will not be audible during online matches.

3. Pilot Version 2
This is a more complex pilot than the version 1. You must to provide an
additional pilot.spt file. With this, the pilot is able to play animations
with sounds. Use Ace Nuko's Brother from the base pilots as example.

Procedure
　1: The first three step is the same as in the version 1.
　2: Create an additional pilot.txt, and rename it to pilot.spt.
　3: Write the animation script in the pilot.spt file and save it.
　　※The PilotEditor can create pilot.spt files with a more simple way than manually.
