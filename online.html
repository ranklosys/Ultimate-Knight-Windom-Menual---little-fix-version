Online mode

Either by connecting to an existing server or by starting a
new one, you are able to play others through the internet.

You can start your own server by using the WindomXP Server.exe
located within the "Server" folder.

・Connecting to servers
Procedures
　１．Select the Online Battle Mode to open the connection screen.
　
　２．Enter the IP Address of the server you want to connect, or you
　　　can use the saved addresses from the list. See below how to
　　　save addresses in the list.
　　　
　３．Enter the name
　　　This name will be displayed during online match. A save data
　　　will be created with the same name. Anytime you want to play
　　　online, you can select the name from the list.
　　　
　４．Select the pilot and the mech you want to use.
　　　The mech can be changed in the lobby too, but the pilot isn't.
　　　
　５．Click on the LOGIN button.
　
　６．At the room selection screen you're may able to choose from
　　　five room. Select the one you want to enter.
　　　
　７．After you entered in a room, you can set yourself to ready by
　　　pressing the F1 button. The match start when all players are
　　　set to ready. It is requires some time to prepare. 　　　

・Online save data
After a name used to login, the save data will be created.
Any number of save data can be created.

By fighting online, the achived score becomes experience. You can win
or lose experience, and get ranks depending on you experience points.
The ranks has no effects on anything, just show the combat potencial.

・You gain an additional 20000 point bonus if you win, and deduct 10000
　if you lose the match.
・If the match will ends in two minutes, you will not gain experience.
・Quitting from the game before the match ends, you lose 20000 points.


Editing the server list
　Open the "Server.ini" located inside of the "Game" folder with any text
　editor. You can add any server to the list using the following example.
	[ServerList]
	Cnt=2
	S0_IP=alexholth.dyndns.org
	S0_Name=Default Server
	S1_IP=127.0.0.1
	S1_Name=Local
　
　Add the address like this. 　
	[ServerList]
	Cnt=3
	S0_IP=alexholth.dyndns.org
	S0_Name=Default Server
	S1_IP=127.0.0.1
	S1_Name=Local
	S2_Name=Friend's Server
	S2_IP=123.210.2.15
　Cnt stands for the quantity of the listed servers. It start from 0.
　Since here are already a S0 and a S1, you need to use the S2 tag.
　S2_Name is the name of the server. You can use any name.
　S2_IP is the IP address of the computer used to host the server.
　
　The next server on the list would be like S3_Name and S3_IP.
　　
　
・Attention
　Players with weaker PC performance can generate slowdowns. It would
　affects all the players in the same room.
　Be sure to adjust the game settings during the start-up to make the
　game run smoothly on your PC.


Setting up server
　Before starting the server, open the TCP ports 2362-2366, and 3362-3366.
　Do not setup a server for your own. If you want to play with others,
　connect to their servers, so you don't need to open the ports.
